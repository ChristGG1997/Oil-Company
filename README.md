# Soil Sales Company

Stock donde pueden acceder usuarios creados y asignados por el admin para tener el control y un historial de todas las compras y ventas realizadas por esta compañia

## Comenzando 🚀

localhost:7000/

### Pre-requisitos 📋

NodeJS v12.10+
mySQL v14.14+

### Instalación 🔧

--> Descargar repo
--> npm install -g
--> npm run dev
